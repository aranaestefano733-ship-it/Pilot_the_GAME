# Pilot_the_GAME
Este es el primer prototipo de un videojuego...

Descripcion:
Videojuego 2D de 5 niveles en el que se manejaran mecanicas de fisica especulativa, movimientos tipo proyectil, transformaciones y carrera.

Sinopsis:
El dron espacial XDrr30t_pb1, apodado Artherium, cae en un planeta aislado mientras transporta un cargamento esencial para la nave principal.
La IA integrada del dispositivo activa el protocolo de adaptación, otorgándole diversas transformaciones que le permitirán sortear los obstáculos de este nuevo mundo. A medida que explora el entorno, su objetivo es mantener el cargamento intacto y encontrar la manera de regresar a la nave nodriza.

Etapas/Niveles:

1. Mundo Subterraneo: Repleto de tuneles laberientosos, zonas de inundacion, erupciones de magma, etc. 
- La movilidad es una carrera contra el tiempo realizando Parkour mientras Arthur escapa del colapso recurrente de los tuneles principales.

2. Pre-Terra: En esta etapa se explora un ligero vistaso del ecosistema, primero una mega flora con arboles gigantescos de los cuales solo se dibisan las raicez y tallo.
- Para esta etapa se realisan interacciones con el ecosistema como enemigo. Para cumplir el objetivo Artherium tendra que mantener el sigilo, movimientos veloces y camuflaje.

3. Ascencion por la jungla: En esta el dron Arthur debe llegar hasta la copa del arbol mas alto para poder divisar una ruta de escape y contactar con la nave nodriza.
- La movilidad cambia para proporcionar retos de escalada.

4. Persecucion: A pocos metros de llegar al objetivo la carga es hurtada por una especie hominida curiosa, queda poco tiempo y Artherium debe recuperarlo cuanto antes. Las mecanicas se transforman de una exploracion y platafrom tranquilas a adoptar dinamicas de combate y caza frenetica.

5. Combate aereo:mEn este nivel Artherium debe evadir a los depredadores aereos Apex en este extraño ecosistema mientra llega al punto de en cuentro para que un modulo de recuperacion lo intercepte
